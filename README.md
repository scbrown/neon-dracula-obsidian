```
Intel One Mono → Fira Code → JetBrains Mono → monospace
```

---

## Install

### From Obsidian

```
Settings → Appearance → Manage → Search "Neon Dracula" → Install → Use
```

### Manual

```bash
cd your-vault/.obsidian/themes
mkdir "Neon Dracula"
# Copy theme.css and manifest.json into folder
```
